# 03. E-Commerce Sales – Advanced Interactive Dashboard

An advanced Power BI solution built on the same e-commerce dataset, focusing on interactivity, user experience, and deeper analytical insights.

---

## 🎯 Objective

To provide decision-makers with an interactive dashboard that enables deeper exploration of sales performance, profitability, customer behavior, and retention trends.

---

## 📌 What’s Included

- Power BI Desktop report file (`ecommerce_sales_v1.pbix`)
- Multi-page report (Home & Dashboard)
- Advanced DAX measures with time intelligence
- Organized measures using Display Folders
- Interactive navigation using bookmarks and toggle buttons

---

## 📊 Highlights

- KPI comparison vs previous year
- YOY / MOM / QOQ analysis
- Toggle-based visuals for multiple analytical views
- Retention, shipping, top products, and top customers analysis
- Enhanced dashboard UX and visual consistency
