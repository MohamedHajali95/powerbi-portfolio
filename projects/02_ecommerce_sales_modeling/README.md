# 02. E-Commerce Sales – Data Modeling & Analysis

A Power BI project focused on data preparation, normalization, and analytical modeling using an e-commerce sales dataset.

---

## 🎯 Objective

To build a clean and reliable analytical model that supports accurate reporting and future dashboard development.

---

## 📌 What’s Included

- Power BI Desktop report file (`ecommerce_sales_v0.pbix`)
- Data preparation using Power Query
- Normalized data structure
- Star schema data model
- Dedicated Calendar table
- Core analytical DAX measures

---

## 📊 Highlights

- Removal of duplicate and empty records
- Correct data types and structured relationships
- Fact and Dimension tables following best practices
- Foundation for scalable and advanced analysis
