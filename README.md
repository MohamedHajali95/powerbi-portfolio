## 🚀 Projects Overview

### 01️⃣ Cafe Sales Dashboard  
📁 `projects/01_cafe_sales_dashboard`

An introductory Power BI dashboard created to present café sales performance to management.

**Highlights**
- Core KPIs (Sales, Profit, Cost, Quantity)
- Interactive slicers (Branch, Product, Payment Method)
- Monthly sales trends
- Management-level overview

**Focus**
- Understanding the analytical workflow  
- Translating raw data into clear business visuals  

---

### 02️⃣ E-Commerce Sales – Data Modeling & Analysis  
📁 `projects/02_ecommerce_sales_modeling`

A deeper analytical project emphasizing **data preparation and modeling best practices**.

**Highlights**
- Data cleaning & normalization
- Star schema (Fact & Dimension tables)
- Dedicated Calendar table
- Structured DAX measures
- Foundational analytical dashboards

**Focus**
- Power Query transformations  
- Data modeling discipline  
- Analytical accuracy  

---

### 03️⃣ E-Commerce Sales – Advanced Interactive Dashboard  
📁 `projects/03_ecommerce_advanced_dashboard`

An advanced Power BI solution built on the same e-commerce dataset, focusing on **interactivity, UX, and deeper insights**.

**Highlights**
- Multi-page report (Home + Dashboard)
- KPI comparison vs previous year
- Advanced time intelligence (YOY, MOM, QOQ)
- Bookmarks & toggle buttons
- Multiple analytical views within the same visual space
- Retention, shipping, top products, and customer analysis
